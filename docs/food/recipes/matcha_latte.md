# Matcha Latte

Serving size: 200/300 ml

## Macros

|  | Carbs (g) | Proteins (g) | Fats (g) | Cals (kcal) |
| --- | --- | --- | --- | --- |
|  |  |  |  |  |
|  |  |  |  |  |

## Ingredients

- 100/200 ml milk
- 100 ml brewed matcha tea
- 50 g Ice (Optional)

## Kitchen Items

- Glass

## Preparation

- Add ice to glass
- Add milk
- Slowly add matcha tea over drink
- Stir and enjoy!
